**Query**: Fetch the inventory variances of the products where the reason is ‘VAR_LOST’ or VAR_DAMAGED.

**Solution**:
```sql
select
	product_id,
	variance_reason_id
from
	Inventory_Item_Variance
join Inventory_Item
		using (inventory_item_id)
where
	variance_reason_id in ("VAR_LOST", "VAR_DAMAGED");
```
