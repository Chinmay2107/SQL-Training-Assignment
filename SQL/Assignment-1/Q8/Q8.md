**Query**: Orders that have more than one item in a single ship group:

**Solution**:
```sql
SELECT
	order_id, COUNT(order_item_seq_id) as total_count 
FROM
	order_item_ship_group_assoc
GROUP BY
	order_id,
	ship_group_seq_id
HAVING
	total_count>1;
```
