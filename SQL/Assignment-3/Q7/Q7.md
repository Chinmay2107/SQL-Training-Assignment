**Query**: On a city-wise basis, what is the analysis of returns?<br>

**Solution**:
```sql
SELECT
	pa.CITY, COUNT(return_id) as Count
FROM
	Return_Header
JOIN Return_Contact_Mech
		using (RETURN_ID)
JOIN Postal_Address pa
		using (CONTACT_MECH_ID)
GROUP BY pa.CITY
ORDER BY Count DESC;
```
